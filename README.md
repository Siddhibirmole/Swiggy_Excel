📊 Swiggy Sales Dashboard | Excel Data Analytics Project

This project is an interactive Swiggy Sales Dashboard built using Microsoft Excel, designed to analyze food delivery data and extract meaningful business insights.

The dashboard transforms raw data into visual reports to understand sales performance, customer behavior, and regional trends. It provides a clear view of key business metrics such as total sales, average order value, customer ratings, and order distribution across different categories.

Through this project, I applied data analytics techniques like data cleaning, pivot tables, pivot charts, Power Query, and interactive slicers to build a structured and dynamic dashboard.

🔍 Key Insights Covered:
Monthly and weekly sales trends
Daily order and revenue patterns
Sales comparison (Veg vs Non-Veg)
State-wise sales distribution (Map visualization)
Quarterly performance analysis
Top 5 cities by sales contribution
🛠️ Tools & Technologies:

Microsoft Excel | Pivot Tables | Pivot Charts | Power Query | Slicers | Conditional Formatting | Data Visualization

🎯 Objective:

To convert raw sales data into actionable insights that help in understanding business performance and improving decision-making using data visualization techniques.

📌 Note: Raw dataset is large, so it is not included in this repository; only dashboard files are provided.
